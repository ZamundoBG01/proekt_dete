=== УРОК 2: ПРАКТИКА С C И УКАЗАТЕЛИ В RAM ПАМЕТТА ===

1. КОД НА ЕЗИКА C:
   int number = 42;       // Създаваме променлива с стойност 42
   int *ptr = &number;    // ptr съдържа АДРЕСА на number в RAM (например 0x7FFF001)

2. КАК ИЗГЛЕЖДА RAM ПАМЕТТА (СИМУЛАЦИЯ):
   [Адрес: 0x7FFF001] -> Стойност: 42 (Променлива 'number')
   [Адрес: 0x7FFF008] -> Стойност: 0x7FFF001 (Указател 'ptr', сочещ към 'number')

3. ДЕРЕФЕРЕНЦИРАНЕ (Вземане на стойността през адрес):
   *ptr = 100; // Променяме стойността на адреса, така че 'number' вече става 100!

4. ГРАНИЧНИ УСЛОВИЯ И БЕЗОПАСНОСТ В ПАМЕТТА:
   - Ако указателят сочи към несъществуващ адрес (NULL или 0x00000000), опитът да се прочете/запише води до срив (Segmentation Fault).
   - Приоритет при писане на ниско ниво: Винаги се проверява дали указателят е валиден преди използване!
